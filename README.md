# Students Grades Web Service API

</header>

<section id="general">

This is a web application of student grades, the details are returned as JSON objects.  
Each student has id, name, grade, couese and year. Here is an example:  

{  
  "id" : "1",  
  "name" : "Elsa",  
  "grade" : "78",  
  "course" : "ws",  
  "year" : "2015"  
}

</section>
